Automated Cover Letter Generator

Python-based Document Automation Project

An efficient Python-based tool that automates the creation of professional cover letters. This project leverages ReportLab for PDF generation, incorporating automated date insertion, customizable personal and recipient details, and a structured layout with headers, body text, and closing sections. The tool streamlines the job application process by generating consistently formatted, professional cover letters.
